# Smart Task Manager

A modular, production-style Task Manager built with **Vanilla JavaScript (ES6+)** demonstrating advanced core JavaScript concepts including closures, OOP, event delegation, and local storage persistence.

This project was built to showcase deep understanding of JavaScript fundamentals without relying on frameworks.

[netlify](https://69975c35e186291539d1bfc2--spiffy-semolina-5c9e5b.netlify.app/)


## Features

- Add / Delete tasks
- Mark tasks as complete
- Filter tasks (All / Completed / Pending)
- Search tasks (with debouncing)
- Persistent storage using localStorage
- Live task statistics (using reduce)
- Dark / Light theme toggle
- Modular file structure

## How to Run

Because this project uses ES6 modules, it must run on a local server.

### Option 1: VS Code Live Server
1. Install Live Server extension
2. Right-click `index.html`
3. Click "Open with Live Server"

### Option 2: Node

```bash
npx serve

```
